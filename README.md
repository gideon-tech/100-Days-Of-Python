# Band Name Generator

A simple Python project that generates creative band names based on user input.

## Description

This project takes two pieces of information from the user:
1. The city where they grew up
2. The name of a pet

It then combines these inputs to create a unique band name suggestion.

## How to Run

1. Make sure you have Python installed on your system
2. Navigate to the project directory
3. Run the script:
   ```bash
   python main.py
   ```

## Example Usage

```
Welcome to the Band Name Generator.
Which city did you grow up in?
Nashville
What is the name of a pet?
Buddy
Your band name could be Nashville Buddy
```
